## windows下音量控制python脚本
python版本:2.7
带客户端，服务端，不支持外网，使用前设置客户端中的ip。
* 默认端口：8017
* 声音开关：0或1
* 增大音量：+
* 减小音量：-
* 关机：-1
* 发送命令行命令：'cmd+空格'+命令

## Windows volume control python scripts
Python version: 2.7
set the IP in client.py before use.
* the default port: 8017
* voice switch: 0 or 1
* volume+: +
* volume-: -
* shutdown: -1
* send command: 'cmd + space' + command  
```cmd shutdown -s -t 100```
